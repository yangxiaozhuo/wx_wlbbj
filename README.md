第一步：下载依赖组件库
```shell
# 通过 npm 安装
npm i @vant/weapp -S --production

# 通过 yarn 安装
yarn add @vant/weapp --production

# 安装 0.x 版本
npm i vant-weapp -S --production
```

第二步：微信开发者工具->工具->构建npm

第三步：编译运行即可
